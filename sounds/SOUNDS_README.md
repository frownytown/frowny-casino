# Halloween Slot Machine Sound Effects

## Required Audio Files

Place the following audio files (MP3 or WAV format) in this directory:

### Required Audio Files

- **`background-music.mp3`** - Looping Halloween background music
  - Recommended length: 30-60 seconds
  - Should be a spooky, atmospheric track
  - Will play at 30% volume in a loop
  - Examples: haunted house ambience, spooky carnival music, eerie Halloween theme

- **`victory.mp3`** - Victory sound when player wins
  - Recommended length: 2-5 seconds
  - Celebratory Halloween-themed sound
  - Examples: witch cackle, spooky fanfare, evil laugh, "Happy Halloween!"

### Game Sound Effects (Built-in)

The following sounds are generated using simple synthesizer tones and don't require audio files:
- **Spin sound** - High beep when reels start spinning
- **Reel stop sound** - Click sound when each reel stops
- **Lose sound** - Low tone when player doesn't win

## Where to Find Free Halloween Sound Effects

1. **Freesound.org** - https://freesound.org (search for "halloween", "witch cackle", "spooky fanfare", etc.)
2. **Zapsplat** - https://www.zapsplat.com (free with attribution)
3. **Pixabay** - https://pixabay.com/sound-effects/search/halloween/
4. **Mixkit** - https://mixkit.co/free-sound-effects/halloween/

## Tips

- Keep files under 200KB each for fast loading
- MP3 format is widely supported and recommended
- Background music should loop seamlessly (fade in/out at ends)
- Victory sound should be punchy and celebratory
